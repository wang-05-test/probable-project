# probable-project
软件测试学习与作业仓库
## 项目简介
本仓库用于记录和存储软件测试相关的学习资料、课程作业及练习项目，方便个人复习与知识沉淀。
## 已包含内容
软件测试（一）.emmx ：软件测试基础概念与流程的思维导图（包含缺陷管理、测试用例设计等核心知识点）
## 使用说明
可直接下载  .emmx  文件，使用 XMind 或兼容思维导图工具打开查看。
后续将逐步补充测试用例、缺陷报告、自动化测试脚本等内容。
## 学习目标
1. 掌握软件测试基础理论与流程
2. 熟练使用缺陷管理工具与版本控制工具
3. 能够设计并执行完整的测试用例
4. 初步了解自动化测试技术
